# grafana-example

### Run Application
- `docker-compose up -d `  to run the application and start the grafana and prometheus


### Stop application with Grafana on local
- `docker-compose down`

### Simple Check API
- `http://localhost:3000/check`


### Grafana Dashboard view
- `http://localhost:500`
